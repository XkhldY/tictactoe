# TicTacToe


- Tictactoe basic game in React.
- Scoreboard using django rest api to track the winners.
- Play against random choice AI for simplicity.

### Prerequisites:
- Python (>=3.7)
- npm (latest)


### 1- Frontend setup

```
$ cd frontend
$ npm install
$ npm start run
```

### 2- Backend setup

```
$ pipenv install
$ pipenv shell
$ cd backend
$ python manage.py runserver
```
